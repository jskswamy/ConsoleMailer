CommandLine Utility to send email using SMTP protocol 

-f -from           From address
-t -to             To address seperated by comma
-c -cc             Cc address seperated by comma
-b -bcc            Bcc address seperated by comma
-s -subject        Subject for the email
-m -body           Email body
-a -attachment     Files to be attached seperated by comma
-l -configration   Mail client configuration file
-g -generate       Generates sample configuration file
